# glaz-boga
Глаз Бога — мощный OSINT-бот для поиска людей и пробива данных. Ищет по номеру телефона, нику, email и соцсетям, показывает оператора и возможные привязки, сверяет базы утечек и открытые источники. Быстрый запуск в Telegram, удобный интерфейс, анонимность, точные результаты для проверки личности и безопасности.
